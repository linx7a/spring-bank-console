# spring-bank-console

Консольное банковское приложение на Java с использованием Spring Framework (Spring Core, без автоконфигураций). Позволяет создавать пользователей, открывать и закрывать счета, пополнять и снимать средства, переводить деньги между счетами (в том числе с комиссией при переводе между разными пользователями).

Проект выполнен в рамках курса по Spring Core от Sorokin School.

## Технологии

- Java 18
- Spring Context 7 (Spring Core, DI через конструкторы)
- Maven
- JUnit 5

## Функциональность

Приложение поддерживает следующие операции через консоль:

- `USER_CREATE` - создание нового пользователя; автоматически открывается один счёт с начальным балансом
- `SHOW_ALL_USERS` - вывод списка всех пользователей и их счетов
- `ACCOUNT_CREATE` - открытие дополнительного счёта для существующего пользователя
- `ACCOUNT_DEPOSIT` - пополнение счёта
- `ACCOUNT_WITHDRAW` - снятие средств со счёта
- `ACCOUNT_TRANSFER` - перевод между счетами (комиссия взимается только при переводе между счетами разных пользователей)
- `ACCOUNT_CLOSE` - закрытие счёта с переводом остатка на первый счёт пользователя (нельзя закрыть последний счёт)
- `EXIT` - завершение работы приложения

Настройки (стартовый баланс нового счёта и процент комиссии за перевод) задаются в `src/main/resources/application.properties`.

## Как запустить

Требуется JDK 18+ и Maven.

Проще всего запустить класс `linx7a.Main` прямо из IDE (IntelliJ IDEA и т.п.) - Maven-зависимости подтянутся автоматически.

## Структура проекта

```
src/main/java/linx7a/
├── model/       - User, Account
├── service/     - UserService, AccountService (бизнес-логика)
├── command/     - ConsoleOperationType, OperationCommand и его реализации, OperationsConsoleListener
├── config/      - AccountProperties, AppConfig
└── Main.java    - точка входа
```
